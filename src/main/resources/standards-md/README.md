# standards-md

把条文化后的规范文档放在这个目录下。

建议做法：

1. 一个规范一个文件，或者一个章节一个文件
2. 标题层级尽量清晰，方便检索时回溯上下文
3. 一段尽量只讲一个判定规则，避免超长大段文本

推荐格式：

```md
# JTG/T H21-2011

## 6.3 裂缝

### 6.3.2 墩柱裂缝

墩柱出现纵向裂缝时，应根据裂缝宽度、长度、是否渗水等因素进行等级评定。
当裂缝宽度超过 0.2mm 且伴随渗水时，可判为较重病害。
```

这套 `v2` 搜索不会调用向量库，也不会查数据库，
只会扫描这个目录下的 `.md` 文件，并做 grep 风格全文检索。
